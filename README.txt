🎤 華語口語認證暖身與聲音演繹指南

這是一個專為華語教學、口語認證試題暖身及聲音演繹訓練設計的單頁式（Single Page Application）互動練習網站。本工具整合了教育部 2023 - 2025 年正式口語認證試題，並結合 Google Gemini 2.5 Flash TTS API 提供高品質的真人語氣示範。

🌟 五大核心功能

1. 🏠 發音快閃店

題庫涵蓋：收錄三年份單音節字與雙音節詞難字。

隨機挑戰：不重複隨機抽取，適合課堂快速暖身，強化咬字精準度。

2. 👾 語氣百變怪

對話感訓練：精選 7 組考題句子，搭配不同情緒提示（如：溫暖知足、深沉感悟、專業解說等）。

告別機械感：引導練習者根據情境調整語調起伏。

3. 🚀 演說急轉彎

短述架構引導：針對短述大題提供「人物、時間、事件、心得」四格導航。

倒數計時：內建 3 分鐘標準計時器，並提供範文參考與專家建議。

4. 🎯 重點音練習

11 項精修：針對「捲舌元音、翹舌聲母、前後鼻音轉換、兒化韻、連音變化、輕聲、送氣音」進行專項訓練。

視覺化引導：放大字體顯示發音要領與練習字詞，拼音完全遵循詞語連寫規範。

5. 🎭 聲音演繹指南（核心專區）

共鳴腔切換：顱腔、鼻腔、口腔、喉腔、腹腔的生理定位與腳本示範。

自然語氣修煉：建立「對象感」與「邏輯重音位移」的真人語感練習。

四大要素演繹：音色、情感、語調、停頓的步驟化教學與示範。

✨ 真人高音質模式 (API 整合)

本網頁整合了 Gemini 2.5 Flash TTS 引擎，可生成具備呼吸感與情緒張力的示範音訊。

🔒 API 安全保護機制

為確保安全性，本專案不會將 API Key 硬編碼於原始碼中，適合直接部署於 GitHub Pages。

設定金鑰：開啟網頁後，點擊上方「⚙️ API 設定」按鈕。

工作階段儲存：輸入您的 Gemini API Key 後儲存，金鑰僅儲存於您目前的瀏覽器階段（Session），不會上傳至任何伺服器。

獲取金鑰：可至 Google AI Studio 免費申請。

🛠️ 技術細節

前端框架：Tailwind CSS (CDN 引入)

程式邏輯：純原生 JavaScript (Vanilla JS)

語音技術：Web Speech API (標準) + Google Gemini API (高音質)

部署方式：支援 GitHub Pages 靜態部署

🚀 部署教學

將 index.html 上傳至您的 GitHub 儲存庫。

進入儲存庫的 Settings > Pages。

將 Source 設定為 Deploy from a branch，選擇 main 分支後儲存。

幾分鐘後即可透過網址瀏覽您的互動練習工具！
